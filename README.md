## วิธีรันเว็บแอปพลิเคชัน

### วิธีที่ 1: ใช้ VS Code Live Server (แนะนำ)

1. เปิดโฟลเดอร์ `quiz-app` ใน VS Code
2. ติดตั้ง extension "Live Server"
3. คลิกขวาที่ `index.html` > "Open with Live Server"
4. เว็บจะเปิดใน browser ที่ `http://127.0.0.1:5500`

python -m http.server 8000

แล้วเปิด browser ไปที่ `http://localhost:8000`

## รูปแบบไฟล์ JSON

ไฟล์ JSON ต้องมีรูปแบบดังนี้:

```json
{
  "quizTitle": "ชื่อแบบทดสอบ",
  "questions": [
    {
      "question": "คำถามที่ 1",
      "options": ["ตัวเลือก 1", "ตัวเลือก 2", "ตัวเลือก 3", "ตัวเลือก 4"],
      "correctAnswer": 0  // index ของคำตอบที่ถูก (เริ่มจาก 0)
    },
    {
      "question": "คำถามที่ 2",
      "options": ["ตัวเลือก 1", "ตัวเลือก 2", "ตัวเลือก 3", "ตัวเลือก 4"],
      "correctAnswer": 2
    }
  ]
}
```

**หรือรูปแบบ array โดยตรง:**

```json
[
  {
    "question": "คำถามที่ 1",
    "options": ["ตัวเลือก 1", "ตัวเลือก 2", "ตัวเลือก 3", "ตัวเลือก 4"],
    "correctAnswer": 0
  },
  {
    "question": "คำถามที่ 2",
    "options": ["ตัวเลือก 1", "ตัวเลือก 2", "ตัวเลือก 3", "ตัวเลือก 4"],
    "correctAnswer": 2
  }
]
```

**หมายเหตุ:**
- `correctAnswer` คือ index ของตัวเลือกที่ถูก (เริ่มจาก 0)
- ถ้าตัวเลือกที่ถูกคือตัวเลือกแรก ให้ใส่ `0`
- ถ้าตัวเลือกที่ถูกคือตัวเลือกที่สอง ให้ใส่ `1`
- และเช่นเดียวกัน

## การแก้ไขไฟล์ข้อสอบ

เมื่อคุณต้องการแก้ไขไฟล์ข้อสอบ:

1. เปิดไฟล์ `quiz_data.json` ใน text editor
2. แก้ไขข้อสอบ ตัวเลือก หรือคำตอบที่ถูก
3. บันทึกไฟล์
4. Refresh หน้าเว็บ (กด F5) เพื่อโหลดข้อสอบใหม่
